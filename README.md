# 🚀 HiWalk

## 📝 Description 

HiWalk is a dynamic travel platform designed to redefine the way people explore the world. Whether you're a solo adventurer, a group traveler, or a digital nomad, HiWalk connects you with unique destinations, curated experiences, and effortless planning tools;

## 📦 Installation

- Open Terminal
- Run git clone https://github.com/JoaoVicPugliesi/HiWalk.git
- Open the folder 
```
cd hiwalk
```

- ASL/Libras Accessibility
- Travel Booking
- High Contrast
- Dynamic Font Size
- Keyboard Navigation
